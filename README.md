# Runningcoach Bern – Redesign

Interaktive Redesign-Skizze der Website von [Runningcoach Bern](https://www.runningcoach-bern.ch/) (René Böhlen) mit konversionsoptimiertem Aufbau und mehrstufigem Buchungs-Overlay.

## Ansehen

`index.html` im Browser öffnen – alles ist in einer einzigen Datei (CSS, JS und das Foto als Base64 eingebettet), keine Abhängigkeiten.

## Inhalt

- Vollflächiger Hero mit Lauf-Bild und klarem Primär-CTA (kostenloses Erstgespräch)
- Leistungen (1:1, Marathon, Trail) + Trainingsplanung & Lauf-Analysen
- Testimonials, Über René, NZZ-Erwähnung
- Preise, FAQ, Geschenkgutschein
- Mehrstufiges Buchungs-Overlay (Leistung → Format → Datum → Zeit → Kontaktdaten → Bestätigung)
- Responsiv für Mobile optimiert

## Hinweise

- Die Buchung endet in einer Demo-Bestätigung – ohne echtes Backend/Kalender.
- Inhalte und Preise stammen aus der Originalseite; es wurden keine Fakten erfunden.
- `rene-boehlen.jpg` ist die Quelldatei des Fotos und wird von der Seite nicht mehr benötigt (bereits eingebettet) – kann optional entfernt werden.

## GitHub Pages

Repo-Settings → Pages → Branch `main` / Root auswählen. Die Seite wird dann unter der Pages-URL ausgeliefert.
